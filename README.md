# Autonomous SOC Security Agent

An AI-powered Security Operations Center (SOC) assistant built with Python, LangChain, and Ollama.

## Features

- Analyze security logs
- Generate incident reports
- AI-powered threat analysis
- Uses Llama 3 via Ollama
- Built with LangChain

## Tech Stack

- Python
- LangChain
- Ollama
- Llama 3
